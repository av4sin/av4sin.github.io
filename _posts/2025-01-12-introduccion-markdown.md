## ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que te permite escribir contenido formateado usando texto plano. Es perfecto para blogs, documentación y notas.

## Sintaxis básica

### Encabezados

```markdown
# Encabezado 1
## Encabezado 2
### Encabezado 3
```

### Énfasis

```markdown
*cursiva* o _cursiva_
**negrita** o __negrita__
***negrita y cursiva***
~~tachado~~
```

Resultado: *cursiva*, **negrita**, ***negrita y cursiva***, ~~tachado~~

### Listas

**Lista desordenada:**
- Elemento 1
- Elemento 2
  - Subelemento

**Lista ordenada:**
1. Primero
2. Segundo
3. Tercero

### Enlaces e imágenes

```markdown
[Texto del enlace](https://ejemplo.com)
![Texto alternativo](imagen.jpg)
```

### Código

Código en línea: `console.log("Hola")`

Bloque de código:

```javascript
function saludo(nombre) {
    return `¡Hola, ${nombre}!`;
}
```

### Citas

> Esto es una cita.
> Puede tener múltiples líneas.

### Tablas

| Columna 1 | Columna 2 | Columna 3 |
|-----------|-----------|-----------|
| Dato 1    | Dato 2    | Dato 3    |
| Dato 4    | Dato 5    | Dato 6    |

Markdown es simple pero poderoso. Una vez que lo aprendas, escribirás contenido mucho más rápido.
