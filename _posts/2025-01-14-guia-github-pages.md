## ¿Qué es GitHub Pages?

GitHub Pages es un servicio de hosting estático gratuito que te permite publicar sitios web directamente desde un repositorio de GitHub. Es perfecto para:

- **Portfolios personales**
- **Blogs técnicos**
- **Documentación de proyectos**
- **Páginas de aterrizaje**

## Ventajas de usar GitHub Pages

### 1. Gratuito
No tienes que pagar nada por el hosting. GitHub se encarga de todo.

### 2. Integración con Git
Cada cambio que hagas se versiona automáticamente. Puedes revertir cambios fácilmente.

### 3. Dominio personalizado
Puedes usar tu propio dominio si lo deseas.

### 4. HTTPS incluido
Seguridad SSL/TLS sin configuración adicional.

## Cómo empezar

1. Crea un repositorio con el nombre `tunombre.github.io`
2. Añade un archivo `index.html`
3. ¡Listo! Tu sitio estará disponible en `https://tunombre.github.io`

## Usando Jekyll

Jekyll es el generador de sitios estáticos que GitHub Pages usa por defecto. Te permite:

- Escribir contenido en Markdown
- Usar plantillas reutilizables
- Crear blogs fácilmente

```yaml
# Ejemplo de _config.yml
title: Mi Blog
description: Un blog personal
theme: jekyll-theme-yat
```

## Conclusión

GitHub Pages es una excelente opción para hosting gratuito de sitios estáticos. ¡Anímate a probarlo!
